# SmartHome

- My smart home solution by HomeKit.
- 我的智能家庭解决方案。

## 目标

- 达到基本的自动/联动化
- 可以使用 Siri 控制
- 控制预算

## 房间

- 合租房
- 2.4/5 G WiFi

## 情景

- 到家，打开门，打开卧室灯
- 离家，关闭门，关闭卧室灯、台灯
- 晚上 11 点，关闭卧室灯
- 晚上 12 点，关闭所有灯
- 早晨 8 点，打开卧室灯
- 学习模式，卧室灯关闭，台灯打开
